# Задача 2. Моделирование динамики популяций видов

## Описание задачи

Задача состоит в прогнозировании динамики популяций в системе “жертва-хищник” на основе временных рядов. В качестве данных используется датасет Figshare predator-prey cycles, содержащий экспериментальные наблюдения за изменением численности водорослей и коловраток.

Водоросли рассматриваются как ресурс или жертва, а коловратки как потребитель или хищник. Основная цель работы заключается в том, чтобы по предыдущей динамике нескольких биологических показателей спрогнозировать численность обеих популяций на следующий день.

## Датасет

Используется архив:

```text
data/10045976.zip
```

Датасет содержит несколько экспериментальных циклов, каждый из которых представляет отдельный временной ряд. Поэтому циклы не объединялись в один непрерывный ряд, а обрабатывались отдельно.

После подготовки данных:

| Показатель                          |                Значение |
| ----------------------------------- | ----------------------: |
| Количество наблюдений               |                    1963 |
| Количество колонок                  |                       8 |
| Количество экспериментальных циклов |                      10 |
| Временной шаг                       |                  1 день |
| Длина циклов                        | от 87 до 375 наблюдений |

## Переменные

В работе используются следующие признаки:

| Переменная        | Описание                                     |
| ----------------- | -------------------------------------------- |
| `algae`           | численность водорослей, ресурс / жертва      |
| `rotifers`        | численность коловраток, потребитель / хищник |
| `egg_ratio`       | доля / соотношение яиц у коловраток          |
| `eggs`            | количество яиц                               |
| `dead_animals`    | количество погибших особей                   |
| `external_medium` | показатель внешней среды / питательной среды |

Целевые переменные:

```text
algae
rotifers
```

Модель прогнозирует значения этих двух переменных на следующий день.

## Подготовка данных

В ходе подготовки данных были выполнены следующие шаги:

* распаковка архива `10045976.zip`;
* объединение исходных CSV-файлов;
* стандартизация названий колонок;
* обработка пропусков;
* интерполяция временных рядов;
* приведение временной шкалы к дневному шагу;
* сохранение отдельных экспериментальных циклов;
* формирование временных окон для LSTM.

Для модели использовалось окно из 14 предыдущих дней. На основе этих данных строился прогноз на 1 день вперед.

## Модель

Для прогнозирования использовалась LSTM-модель, так как она подходит для анализа временных зависимостей в последовательных данных.

Параметры постановки задачи:

| Параметр                      |                        Значение |
| ----------------------------- | ------------------------------: |
| Тип задачи                    | прогнозирование временного ряда |
| Вход модели                   |                 14 дней истории |
| Количество входных признаков  |                               6 |
| Горизонт прогноза             |                          1 день |
| Количество целевых переменных |                               2 |

Архитектура модели:

```text
LSTM(64, return_sequences=True)
Dropout(0.20)
LSTM(32)
Dropout(0.20)
Dense(16, relu)
Dense(2)
```

Финальный слой `Dense(2)` используется, потому что модель одновременно прогнозирует две связанные популяции: водоросли и коловраток.

## Разделение выборки

Данные были разделены с сохранением временного порядка:

| Часть выборки | Количество окон |
| ------------- | --------------: |
| Train         |            1271 |
| Validation    |             274 |
| Test          |             278 |

Перемешивание данных не использовалось, так как для временных рядов важно сохранять порядок наблюдений.

## Метрики качества

Для оценки качества прогноза использовались:

* MSE, Mean Squared Error;
* MAE, Mean Absolute Error.

Результаты на тестовой выборке:

| Target   | MSE scaled | MAE scaled | MSE original | MAE original |
| -------- | ---------: | ---------: | -----------: | -----------: |
| algae    |     0,0072 |     0,0471 |       0,4762 |       0,3840 |
| rotifers |     0,0093 |     0,0582 |     210,2272 |       8,7679 |
| overall  |     0,0082 |     0,0526 |     105,3517 |       4,5760 |

## Интерпретация результатов

Модель смогла уловить общую динамику изменения численности популяций. Прогноз для водорослей оказался точнее, чем для коловраток. Это связано с тем, что динамика коловраток более нестабильная и содержит резкие всплески численности.

Наиболее сложными для модели оказались участки с резкими изменениями популяции. LSTM-модель в целом повторяет направление динамики, но склонна сглаживать экстремальные пики.

## Описание файлов

* `training_mae.png` — график изменения MAE на train и validation.

## Вывод

LSTM-модель показала, что может использоваться для моделирования взаимосвязанных временных рядов в системе “жертва-хищник”. Она достаточно хорошо отражает общую динамику популяций, однако хуже справляется с резкими скачками численности, особенно для коловраток.
