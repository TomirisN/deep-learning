# Анализ и генерация данных о пищевых цепочках

## Цель работы

Применить методы глубокого обучения для анализа данных о пищевых цепочках, классификации типов взаимодействий между видами, моделирования динамики популяций, а также генерации новых схем пищевых цепочек и текстовых описаний экосистем на основе открытых данных. Разработать систему, способную понимать и создавать элементы пищевых цепочек для экологических исследований.

## Список задач лабораторной работы

1. Сбор и предварительная обработка данных о пищевых цепочках.
2. Классификация типов взаимодействий между видами (хищник-жертва, симбиоз).
3. Моделирование динамики популяций видов в экосистеме.
4. Генерация схем новых пищевых цепочек.
5. Генерация текстовых описаний экосистем.
6. Анализ сетей связей между видами.
7. Оценка качества и реалистичности сгенерированных данных.
8. Анализ результатов и визуализация экспериментов.

## 1. Теория

### 1.1. Пищевые цепочки

Пищевые цепочки — это системы взаимодействий между видами в экосистеме

- Хищник-жертва
- Симбиоз - взаимовыгодное сотрудничество.
- Популяции - численность видов, изменяющаяся во времени.

Эти данные используются для изучения экологии, устойчивости экосистем и влияния внешних факторов.

### 1.2. Особенности анализа данных

- Взаимодействия классифицируются по типу (хищничество, паразитизм), могут быть представлены как пары видов.
- Популяции описываются временными рядами численности видов.
- Схемы цепочек - графические структуры, которые можно представить как изображения или графы.

### 1.3. Методы анализа

1. Классификация взаимодействий. Используются сверточные нейронные сети (CNN) для анализа данных о видах.
2. Моделирование динамики. Применяются LSTM для прогноза изменений популяций.
3. Генерация данных. Используются GAN для схем цепочек и GPT-2 для текстовых описаний.
4. Сетевой анализ. NetworkX для связей между видами.

## 2. Набор данных

### Описание набора данных

- GBIF (Global Biodiversity Information Facility). Данные о видах и их распространении (https://www.gbif.org). Содержит миллионы записей о видах с координатами и характеристиками.
- Predator-Prey Database. Взаимодействия и популяции видов (https://www.globalbioticinteractions.org). Содержит 10,000+ взаимодействий между видами и временные ряды популяций.
- EcoBase. Модели пищевых цепочек и экосистем содержит 200+ моделей пищевых цепочек с данными о биомассе (http://ecobase.ecopath.org).

### Пример данных

| Вид 1 | Вид 2 | Тип взаимодействия | Популяция Вид 1 (тыс.) | Популяция Вид 2 (тыс.) |
| --- | --- | --- | --- | --- |
| Волк | Олень | Хищник-жертва | 5 | 50 |
| Пчела | Цветок | Симбиоз | 100 | 200 |

## Задачи лабораторной работы

### Задача 1. Классификация типов взаимодействий между видами

**Описание:** Использовать данные о видах для определения типа взаимодействия (хищник-жертва, симбиоз, паразитизм).

**Этапы**

1. Загрузить данные из Predator-Prey Database и преобразовать пары видов в признаки (например, численность, биомасса).
2. Построить CNN для классификации.
3. Обучить модель на данных с метками типов взаимодействий.
4. Оценить точность на тестовой выборке.

Ключевая идея заключается в том, что CNN анализирует числовые характеристики взаимодействий.

**Метрики**

Accuracy, Precision, Recall, F1-score.

**Пример структуры модели**

```python
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv1D, MaxPooling1D, Flatten, Dense

model = Sequential([
    Conv1D(32, 3, activation='relu', input_shape=(feature_length, 1)),  # Признаки пар
    MaxPooling1D(2),
    Conv1D(64, 3, activation='relu'),
    MaxPooling1D(2),
    Flatten(),
    Dense(128, activation='relu'),
    Dense(3, activation='softmax')  # 3 типа взаимодействий
])
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])
```

### Задача 2. Моделирование динамики популяций видов

Прогнозировать изменения численности видов в пищевой цепочке на основе временных рядов.

**Этапы**

1. Подготовить данные из EcoBase или Predator-Prey Database (временные ряды популяций).
2. Построить модель LSTM для анализа последовательностей.
3. Обучить модель на исторических данных.
4. Оценить точность прогноза.

Ключевая идея заключается в том, что LSTM моделирует взаимозависимости популяций (например, рост хищников снижает число жертв).

**Метрики**

Mean Squared Error (MSE), Mean Absolute Error (MAE).

**Пример структуры модели**

```python
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense

model = Sequential([
    LSTM(64, input_shape=(time_steps, features), return_sequences=True),
    LSTM(32),
    Dense(1)  # Прогноз численности
])
model.compile(optimizer='adam', loss='mse')
```

### Задача 3. Генерация схем новых пищевых цепочек

Создать модель для генерации реалистичных схем пищевых цепочек (графические изображения).

**Этапы**

1. Построить GAN для генерации изображений.
2. Обучить модель на визуализациях пищевых цепочек из EcoBase (переведенных в изображения).
3. Сгенерировать новые схемы цепочек.
4. Оценить качество сгенерированных изображений.

Ключевая идея заключается в том, что GAN воспроизводит структуру пищевых цепочек.

**Метрики**

Inception Score (IS), Fréchet Inception Distance (FID).

**Пример структуры генератора**

```python
from tensorflow.keras.layers import Dense, Reshape, Conv2DTranspose
from tensorflow.keras.models import Sequential

generator = Sequential([
    Dense(128 * 32 * 32, input_dim=100, activation='relu'),
    Reshape((32, 32, 128)),
    Conv2DTranspose(64, (3, 3), strides=2, padding='same', activation='relu'),
    Conv2DTranspose(3, (3, 3), strides=2, padding='same', activation='sigmoid')  # RGB схема
])
```

### Задача 4. Генерация текстовых описаний экосистем

Создать модель для генерации текстовых описаний пищевых цепочек и экосистем (например, "В лесу волки охотятся на оленей").

**Этапы**

1. Построить GPT-2 и обучить на текстовых данных из GBIF и EcoBase.
2. Сгенерировать новые описания экосистем.
3. Оценить качество текстов.

Ключевая идея заключается в том, что GPT-2 создает связные описания на основе примеров.

**Метрики:** Perplexity, Mean Opinion Score (MOS).

**Пример кода**

```python
from transformers import GPT2LMHeadModel, GPT2Tokenizer

tokenizer = GPT2Tokenizer.from_pretrained('gpt2')
model = GPT2LMHeadModel.from_pretrained('gpt2')

prompt = "In this ecosystem, predators"
inputs = tokenizer(prompt, return_tensors='pt')
outputs = model.generate(**inputs, max_length=50)
print(tokenizer.decode(outputs[0], skip_special_tokens=True))
```

### Задача 5. Анализ сетей связей между видами

**Описание:** Исследовать взаимодействия между видами в пищевых цепочках (хищничество, симбиоз).

**Этапы:**

1. Построить граф на основе данных Predator-Prey Database (узлы: виды, ребра: взаимодействия).
2. Использовать NetworkX для анализа ключевых узлов (например, вершинные хищники).
3. Визуализировать граф.

Ключевая идея заключается в том, что сетевой анализ выявляет структуру экосистемы.

**Пример кода**

```python
import networkx as nx
import matplotlib.pyplot as plt

G = nx.DiGraph()  # Направленный граф для цепочек
G.add_edge("Wolf", "Deer", weight=5)
G.add_edge("Deer", "Grass", weight=10)
nx.draw(G, with_labels=True, arrows=True)
plt.show()
```

## 3. Анализ результатов

- Цель: Оценить точность CNN на данных Predator-Prey Database.
- Метрики: Accuracy, Precision, Recall, F1-score.
- Ожидания: Точность > 80%, сложности с редкими взаимодействиями (например, паразитизм).

- Метрики: MSE, MAE.
- Ожидания: MSE < 0.1, ошибки при резких изменениях (например, вымирание).

- Цель: Оценить качество схем (GAN) и текстов (GPT-2).

- Ожидания: Реалистичные схемы и связные описания.
